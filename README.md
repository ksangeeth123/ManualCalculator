# Simple Java Calculator

A hand-coded Java Swing calculator designed for learning purposes. This application provides a graphical interface for performing basic arithmetic operations with input validation.

## Features

- **Basic Arithmetic**: Supports Addition (+), Subtraction (-), Multiplication (X), and Division (/).
- **Input Validation**:
  - Prevents non-numeric input.
  - Blocks negative number entries.
  - Handles division by zero errors.
- **Graphical User Interface (GUI)**:
  - Built using Java Swing.
  - Responsive digit buttons (0-9) that input into the last focused field.
  - Modern "Segoe UI" typography.
  - Clear (CLR) function to reset inputs and results.

## Requirements

- Java Development Kit (JDK) 8 or higher.

## How to Run

1.  **Compile the source code**:
    ```bash
    javac ManualCalculator.java
    ```

2.  **Run the application**:
    ```bash
    java ManualCalculator
    ```
